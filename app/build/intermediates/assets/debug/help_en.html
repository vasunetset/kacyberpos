Device Introduce：

This printer is a thermal printer that can use 58 mm width paper and support 48 mm printable width,One printable row has 384 point , Per millimeter Can accommodate 8 points , the feeding paper density is 0.125 mm

Demo Introduce:

This example provides three sub-examples, which are print tests, DIY prints, and web page calls the print API.


Print Test：

1.Normal：Click the "Print" button to print the device information list; click "FEED" button, you can adjust the starting position of the next print.

2.Adcance：Select the "Advanced Settings" radio button to edit the title text information, and you can modify its properties. you can also turn off the print service.

DIY Prints：

1.Normal：You can print one-dimensional code, two-dimensional code, and text according to the contents of the input box.

2.Adcance：Select the "Advanced Settings" radio button, then you can edit the width and height of one-dimensional code or two-dimensional code , edit the property of the text.

Note: The default is to print information on non-label paper, select the "label paper" option, the logic of print on label paper will be performed.

Invoke Print：

Click the "Print" button on the web page,then it will call print's API and print the product purchase list.
